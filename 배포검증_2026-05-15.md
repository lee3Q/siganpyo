## 5. 모바일 접속 가이드

### 접속 URL
**https://lee3q.github.io/siganpyo/**

### iPhone (Safari) 홈 화면 추가
1. Safari로 위 URL 접속
2. 공유 버튼(↑) 탭 → "홈 화면에 추가"
3. 앱 이름 "시간표" 확인 → 추가
4. 홈 화면 아이콘으로 바로 실행 가능 (standalone 모드)

### Android (Chrome) 홈 화면 추가
1. Chrome으로 위 URL 접속
2. 메뉴(⋮) → "홈 화면에 추가" 또는 "앱 설치"
3. 아이콘 생성 → 바로 실행

### 오프라인 동작
- 한 번 접속 후 Service Worker 등록되면 오프라인에서도 앱 Shell 로드
- 이전에 방문한 날짜의 데이터는 localStorage 캐시에서 로드 (24시간)
- 새 날짜 데이터는 네트워크 필요 (네트워크-first 전략)

### 매일 사용법
1. 폰에서 "시간표" 앱 실행
2. 오늘 날짜의 학습 계획이 자동 표시
3. 블록 완료 시 탭하여 체크
4. 데이터 갱신: GitHub에 push된 JSON이 자동 반영 (최대 5분 캐시)
